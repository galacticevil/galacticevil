# Galactic Evil portfolio

Planning workspace for Rory Cottle's professional portfolio at `galacticevil.org`.

The approved direction is a modern, responsive portfolio presented through the visual language of a lost 1990s SVGA science-fiction adventure game. It must communicate serious senior engineering and fractional CTO capability first, with restrained genre-aware humour as flavour.

## Current status

Planning and documentation are complete enough to scaffold the application. The Nuxt project has deliberately not yet been created.

## Approved stack

- Nuxt with Vue 3 and TypeScript
- Nuxt UI used selectively for accessibility-heavy primitives
- Tailwind CSS for responsive composition and utilities
- VueUse for browser state and animation lifecycle
- Static generation for SureSupport hosting
- Minimal PHP backend, developed locally with Docker
